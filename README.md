![Bioinformatics vs Computational Biology](Bioinformatics.png).

**Bioinformática** é a aplicação de ferramentas computacionais para coletar, armazenar, analisar, manipular, apresentar e compartilhar dados biológicos.

**Biologia computacional** é uma área de pesquisa que visa a aplicação de conhecimentos em computação e outras áreas na resolução de problemas biológicos em aberto, sendo uma área muito mais nobre e estratégica.
